para escribir un parrafo se usa < p>
para escribir en negrita sin que sea una palabra "clave" para google usamos < b> para escribir en negrita pero que se una palabra "clave" para el navegador usamos < strong>
para escribir una palabra en cursiva usamos: < i>
para poner una linea separadora usamos < hr>
para poner un link usamos < a>
para hacer una lista desordenada usamos: < ul> y < li>
para hacer una lista ordenada usamos: < ol> y < li>
para poner una pagina usamos < img src="nombre.formato>
para poner un formulario usamos: < form>

Si queremos citar un parrafo que halla citado alguien usamos: < blockquote>
y para citar el nombre del que dijo el parrafo es: < cite>
